## ✨ المميزات الرئيسية

### 📊 لوحة تحكم تفاعلية
- إحصائيات مباشرة للميزانية والمصروفات
- عرض بياني للمصروفات حسب التصنيف
- متابعة فورية للمبلغ المتبقي

### 💰 إدارة مالية متكاملة
- تعيين الميزانية الشهرية
- تسجيل المصروفات اليومية
- تصنيف المصروفات (طعام، فواتير، مواصلات، إلخ)
- تتبع تاريخ المصروفات

### 📱 تجربة مستخدم متميزة
- تصميم متجاوب يعمل على جميع الأجهزة
- واجهة مستخدم عربية intuitive
- إشعارات فورية للإجراءات
- ألوان مريحة للعين وتصميم عصري

### 🛠️ تقنيات متطورة
- Chart.js للرسوم البيانية التفاعلية
- Font Awesome للأيقونات
- تصميم CSS3 متقدم مع Flexbox و Grid
- JavaScript ES6 للوظائف التفاعلية

## 📖 كيفية الاستخدام

### 1. تعيين الميزانية
- ادخل إلى تبويب "الميزانية"
- أدخل الميزانية الشهرية المتوقعة
- اضغط على "تعيين الميزانية"

### 2. إضافة المصروفات
- انتقل إلى تبويب "إضافة مصروف"
- أدخل قيمة المصروف
- اختر التصنيف المناسب
- (اختياري) أضف وصفاً للمصروف
- اضغط على "إضافة المصروف"

### 3. متابعة الإحصائيات
- تابع الميزانية المتبقية في البطاقة العلوية
- شاهد توزيع المصروفات في الرسم البياني
- راجع قائمة المصروفات المسجلة
